# ObjectOrientedDesign-Deadwood
A design of the board game Deadwood Studios. Follows an object oriented design approach for Western Washington University's object oriented design course. 
# Usage 
Simply clone the repository into a new Java Project. 
# What I Learned 
- Inheritance is used to group similar objects together and allows for greater code reusability. 
- XML files consist of both tags and text, are used to store data, and allow for greater compatibility.
